## Package Structure

parent folder
```java
package Java_strings;
```

Basic Java Program Syntax

```java
package Java_strings;

public class syntax {
    public static void main(String[] args) {
    }
}
```
Explanation:

1. package Java_strings; groups related Java files
2. class syntax is the class name (same as file name)
3. main() is the entry point of the program

String Example in Java
```java
String name = "Satyam Umrao";
System.out.println(name);
```
Explanation:

1. String name declares a string variable
2. "Satyam Umrao" is a string literal
3. System.out.println() prints output to the console

Output
```text
Satyam Umrao
```
